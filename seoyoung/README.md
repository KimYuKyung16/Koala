### 3. Jeep, Problem, Try
- Keep: 현재 만족하는 부분 / 계속 이어갔으면 하는 부분 <br>
- Problem: 불편한 부분 / 개선이 필요하다고 생각하는 부분 <br>
- Try: Problem에 대한 해결책, 다음 회고 때 판별 가능한 것, 당장 실행가능한 것 <br>
- 목적: 짧은 시간에 구성원의 생각응ㄹ 공유하고, 실행가능하고 측정 가능한 Action을 도축
